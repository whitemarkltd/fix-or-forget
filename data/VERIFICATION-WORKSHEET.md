**Rules of the road**
- Only fields with a public live source can reach `high`: **official repair** (maker's page), **refurbished** (Back Market), **DIY parts** (iFixit), **trade-in** (SellCell).
- **Independent-shop repair has no single authoritative source** — leave it; it stays `medium` at best. Not in this worksheet.
- If a page shows one price, write it once (e.g. `$279`). If it shows a spread, write a range (e.g. `$120–160`).
- Blank = skip for now. Do the sections top-down; they're ordered by impact.
